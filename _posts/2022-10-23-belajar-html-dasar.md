## 4. Tag

### tag `<HTML>`

### bagian `<head>`

- judul halaman
  `<title></title>`

- CSS
  `<style></style>`

- javascript
  `<script></script>`

- metadata
  `<meta></meta>`

### bagian `<body>`

- teks
  `<h1>, <h2>, <h3>, <h4>, <h5>, <h6>, <p>, ...`

- pendukung teks
  `<br>, <hr>, <em>, <strong>, ...`

- gambar
  `<img>`

- hyperlink
  `<a>`

- list (bullets & numbering)
  `<ul>, <ol>, <li>, <dl>, <dt>, <dd>`

- tabel
  `<table>, <thead>, <tbody>, ...`

- form
  `<form>, <input>, <select>, <button>, ...`

- script
  `<script>`

- object
  `<object>`

- grouping
  `<div>, <span>`

- komentar
  `<!-- isi komentar -->`

### struktur tag
